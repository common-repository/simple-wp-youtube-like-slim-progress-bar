Simple WordPress YouTube like slim progress bar inspired from NProgress.js.

== Description ==
Simple WordPress YouTube like slim progress bar inspired from NProgress.js. It's very easy to integrate into WordPress website without coding knowledge. Check the demo in http://vijay.pro/ website(If you missed seeing the loader, refresh once & check). It supports multisite (network) too.

== Installation ==

Very easy to install, similar to rest of the plugins.

1. Download and unzip the plugin page-loader.zip.<br />
2. Copy the unzipped folder in your Plugins directory under wordpress installation. (wp-content/plugins)<br />
3. Activate the plugin through the plugin window in the admin panel.<br />
4. Configure the settings through settings -> YouTube like loader in the admin panel.
